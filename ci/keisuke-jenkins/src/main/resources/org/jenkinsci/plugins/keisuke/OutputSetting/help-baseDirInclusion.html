If you check this checkbox, the source code file paths are described starting with the source directory name.
That is as same as 'keisuke.StepCount' or 'keisuke.DiffCount' with '-path base' option saves the counting result into an output file.
If you uncheck it, the source code file paths do not include the source directory name.
That is as same as 'keisuke.StepCount' or 'keisuke.DiffCount' with '-path sub' option saves the counting result into an output file.
