Specify the top directory of the old source code files to compared differences.
The base directory to specify a relative path is this workspace.(e.g. 'old/src/main/java')
If the top directory is not under the workspace, you should write an absolute path. (e.g. '/var/proj/old/src/main/java')
If the top directory equals the workspace, you should write only '.'(dot) into this text box.
